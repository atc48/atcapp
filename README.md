ATC48
====

Map view client for Air Traffic Controller especially in RJJJ FIR Region.

## Requirement

- Modern Web Browser (HTML, CSS, Javascript)
- Python3

## Usage

```
$ python3 make_one_file_html.py > atc48.html
$ open atc48.html
```
